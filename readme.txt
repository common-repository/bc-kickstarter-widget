KickStarter project information directly on your site

== Description ==

"BC Kickstarter widget" allows you to pull single Kickstarter project on your site using widget or shortcode.
Comes with simple style but you can restyle it as you desire.

Promote your project on your Wordpress website and help drive traffic to your Kickstater page.
- Show project image and short description
- Displays the number of backers, current amount funded and percentage funded.
- Customizable widget title name

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Go to Appearance/Widgets, find widget "BC Kickstarter widget" and add to desired sidebar 
4. You can use shortcode instead of a widget: [kickstarter url= &quot;your project url &quot; ]
